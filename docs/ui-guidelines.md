- invoice UI

- clean spacing

- responsive layout
- strong hierarchy

# Table Design Rules

- medium borders
- proper padding
- aligned text
- bold totals
- readable spacing

---

# Animation Rules

Allowed:
- fade in
- slide up
- soft hover effects

Avoid:
- excessive motion
- flashy effects
- distracting animations

---

# Mobile First Rules

- touch friendly
- large buttons
- responsive cards
- sticky actions

---

# Action Buttons

Required Buttons:
- Download PDF
- Share Quotation
- WhatsApp Share
- Save Quotation

---

# Branding Philosophy

- trustworthy
- organized
